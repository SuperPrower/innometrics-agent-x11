#+AUTHOR: Sergey Koziakov

* Building
** Requirements
- Qt5
- xcb and xcb_util-wm
- sqlite

** Compiling
- Create a build directory. =cd= into it:
#+BEGIN_SRC sh
mkdir build && cd build
#+END_SRC

2. Call =qmake= referencing the =.pro= file:
#+BEGIN_SRC sh
qmake ../x-innometrics.pro
#+END_SRC

3. Build the application
#+BEGIN_SRC sh
make
#+END_SRC
